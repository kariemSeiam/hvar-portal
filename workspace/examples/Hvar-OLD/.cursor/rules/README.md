# HVAR Project - Cursor Rules System

## Project-Specific Patterns

### Frontend (React + Vite + Tailwind)
- **Component Organization**: `src/components/` with feature-based subdirectories
- **State Management**: Context-based global state with local state patterns
- **Routing**: Lazy-loaded routes with Suspense fallbacks
- **Styling**: Tailwind CSS with custom design system variables

### Backend (Flask + SQLite)
- **App Factory**: Blueprint-based route organization
- **Service Layer**: Business logic separation from routes
- **Database**: SQLite with connection management patterns
- **API Responses**: Consistent JSON response format

### Design System
- **Color Palette**: Brand red, blue, and accent color system
- **Typography**: Fluid typography with clamp values
- **Spacing**: Fluid spacing system with clamp values
- **Components**: Consistent component variants and patterns

# ⚡ Quick Start

**3 steps. 2 minutes. One command.**

---

## Step 1: Copy

```bash
# Copy these to your project root:
.cursorrules
.cursorignore
.cursor/          # The whole folder
```

## Step 2: Open

```bash
cursor your-project/
```

## Step 3: Talk to DNA

Press `Cmd+L` (Mac) or `Ctrl+L` (Win), then type:

```
/dna scan
```

**Done.** DNA now knows your project.

---

## One Command. Everything.

```
/dna                              # Show help
/dna scan                         # Extract design from code
/dna page dashboard analytics     # Create a page
/dna component button             # Create a component  
/dna feature user-profile         # Create full feature
/dna check                        # Validate code
/dna plan new auth system         # Plan before building
/dna refactor user service        # Safe refactoring
/dna what colors do we use?       # Ask anything
/dna make it darker               # Adapt the style
```

Or just talk naturally:

```
/dna make me a settings page
/dna is this code correct?
/dna create a modal with form
/dna why do we structure it this way?
```

# Start-chat prompt

Paste at the start of a new chat to load context and discussion mode.

---

## Full

```
Working on HUB-MCRM (call center / full project).

Read: dev/MANAGEMENT.md, dev/CONTEXT.md. Docs: docs/INDEX.md (canonical).

Use discussion mode: recognize context changes (decisions, ideas, plan, design), suggest which files to update, show proposed changes, ask for confirmation (Yes/No/Modify) before editing.

Canonical docs: docs/INDEX.md. Acknowledge when ready.
```

---

## Short

```
HUB-MCRM. Read dev/MANAGEMENT.md + dev/CONTEXT.md. Discussion mode: suggest updates, confirm before editing. Docs: docs/INDEX.md. Ready?
```

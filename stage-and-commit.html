Design System / Feedback Components

التنبيهات والإشعارات

كل رسالة للمستخدم هي فرصة — مش مجرد معلومة. الإشعار الصح في المكان الصح، بالـ copy الصح، هو اللي بيفرق بين تجربة مزعجة وتجربة بتبني ثقة. ٨ categories. صفر hardcoded colors. RTL-first.

Alert Banners

.alert

بانرات ثابتة جوه تدفق الصفحة — مش مؤقتة. للرسائل اللي المستخدمة محتاجة تشوفها وتعمل حاجة بناءً عليها.

success
طلبك اتسجل بنجاح ✓
هيوصلك رسالة SMS بتفاصيل الطلب وموعد التسليم المتوقع. شكراً لاختيارك هفار.
error — مع action
في مشكلة في الاتصال
تأكدي من النت وجربي تاني. لو المشكلة فاضلت كلمينا على واتساب.
warning — session expiry
الجلسة هتنتهي قريب
سلتك محفوظة، بس لازم تسجلي دخولك تاني عشان تكملي الطلب.
info — dismissible
شحن مجاني على أوردرات فوق ٢٠٠٠ جنيه
ضيفي قطعة للسلة وهتوفري تكلفة الشحن تلقائياً.
brand — عرض خاص
خصم ١٥٪ على أجهزة الطهي — اليومين دول بس
استخدمي كود COOK15 عند الدفع. الكود شغال على المنتجات المختارة فقط.
/* Variants */ .alert-success → trust-green palette | طلب ناجح، تأكيد .alert-error → hvar-red palette | خطأ، فشل، رفض .alert-warning → amber palette | تحذير، انتهاء جلسة، مخزون .alert-info → blue palette | معلومة، تتبع، شحن .alert-brand → hvar-red 50/200/600 | عروض، إعلانات هفار /* ARIA */ role="alert" → errors & critical warnings (interrupts) role="status" → info & success (non-interrupting)

System Error Cards

.error-card

للأخطاء على مستوى الصفحة. قاعدة صارمة: كل error card لازم فيها مسار تاني — retry، أو واتساب، أو رجوع. مش بس رسالة خطأ وبس. مش "Error 500". مش صفحة بيضا.

في مشكلة في الاتصال
تأكدي من النت وجربي تاني. لو المشكلة فاضلت كلمينا.
الدفع لم يتم
تأكدي من بيانات البطاقة أو اختاري الدفع عند الاستلام وما تخسريش الطلب.
انتهت الجلسة
سجلي دخولك تاني. سلتك محفوظة وما راحتش.
في مشكلة مؤقتة
فريق هفار شايف المشكلة وبيحلها. جربي تاني بعد شوية.

Empty States

.empty

اللحظة الفاضية مش نهاية — هي فرصة. كل empty state بتقول "مفيش حاجة هنا" وبتدي مسار واحد واضح للخطوة الجاية. صوت هادئ ومباشر — مش معتذر ومش زيادة كيوت.

سلتك فاضية
ما فيهاش حاجة لسه. ابدئي التسوق واختاري اللي عاجبك.
مفيش طلبات لحد دلوقتي
أول ما تطلبي، كل طلباتك هتبقى هنا مع تتبع التوصيل.
مفيش تذاكر خدمة
عندك منتج محتاج صيانة أو استبدال؟ ابدئي هنا.
مفيش نتايج لـ "هواء"
جربي كلمة تانية أو استعرضي الفئات المتاحة.
/* Rules — per portal/ux-patterns.md §14 */ // Single CTA — never multiple options // Warm authority voice: not "لا توجد بيانات" → "سلتك فاضية" // Small icon or illustration — no stock photos // Never apologetic, never cute-overloaded
